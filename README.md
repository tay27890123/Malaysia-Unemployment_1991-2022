# Malaysia Youth Unemployment Analysis (2019–2025)

**Bachelor of Economics Project**  
University Malaya

### Objective
Analyze youth unemployment trends in Malaysia before and after COVID-19 using official DOSM data, with focus on age groups and recovery patterns.

###  Tools
- Python 3
- Pandas, NumPy
- Matplotlib & Seaborn
- Plotly (interactive charts)

###  Project Structure
- `notebooks/` → Main analysis
- `data/` → Raw datasets from DOSM
- `images/` → Charts for README and report

###  Current Status
**In Progress** — EDA + Visualization + Econometric Analysis
